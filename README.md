# DASH-COMPRAS

Dashboard estática de acompanhamento operacional da planilha **Notas fiscais sem pedido**.

## Arquivos

- `dashboard-notas-fiscais-sem-pedido.html` — dashboard gerada pronta para publicação
- `scripts/gerar-dashboard-notas-sem-pedido.js` — script gerador da dashboard

## Fonte de dados

Google Sheets: **Notas fiscais sem pedido**

## Observação

A dashboard é gerada localmente com leitura da planilha e pode ser publicada em GitHub Pages, Netlify ou outro host estático.
